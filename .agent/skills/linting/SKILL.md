# Linting Standards

**Requirement**: Ensure all code is cleanly formatted and passes lint checks before considering a task complete.

## Frontend (web-client)
- **Tool**: ESLint
- **Command**: `npm run lint`

## Backend (backend)
- **Tool**: Ruff
- **Command**: `ruff check .` or `ruff format .`
